# Student Course Management System

A comprehensive REST API system for managing students, courses, enrollments, and grades built with Laravel framework.

## Project Overview

This project demonstrates modern web development practices by implementing a complete Student Course Management System with:

- 10 RESTful API endpoints
- Laravel framework with Eloquent ORM
- MySQL database with proper relationships
- Bearer token authentication
- Docker containerization
- Automated deployment scripts

## Key Features

### Student Management
Create, read, update, and delete student records with comprehensive information tracking.

### Course Management
Manage course catalog with detailed course information.

### Enrollment System
Handle student course enrollments with proper relationship tracking between students and courses.

### Grade Tracking
Record and manage student grades for enrolled courses with secure access controls.

### Security
Bearer token authentication protects sensitive enrollment and grade data.

## Technology Stack

- **Backend Framework**: Laravel 11.x
- **Database**: MySQL 9.4.0
- **Containerization**: Docker with Docker Compose
- **Web Server**: Nginx
- **Documentation**: Hugo Static Site Generator
- **Deployment**: GitHub Pages with automated CI/C
